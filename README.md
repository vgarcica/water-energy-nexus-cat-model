# water-energy-nexus-cat-model
Model of the Catalonia water–energy nexus for desalination planning and drought resilience analysis.

1. Tots els fitxers de dades han d'estar al mateix directori que els fitxers de codi.
2. Executar primer el contingut del "launcher.py" que carregarà les dades i les funcions necessàries.
3. A partir d'aquí hi ha varies opcions
4. Explorar escenaris mitjançant el RGS o el NSGAII amb rgs.py o nsgaii.py respectivament
5. Analitzar un escenari concret i graficar el seu perfil horari
6. Generar perfils de la corva d'embassaments per diverses configuracions operatives
